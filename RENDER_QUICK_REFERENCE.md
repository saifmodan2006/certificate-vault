# Render Deployment: Quick Reference

**Fast reference for deploying CertiVault to Render**

## 🚀 3-Minute Deploy (Recommended)

1. **Go to Render Dashboard**
   ```
   https://dashboard.render.com
   ```

2. **Click "New +" → "Root Directory"**

3. **Select Repository**
   - Select: `saifmodan2006/certificate-vault`

4. **Done!** ✅
   - Render reads `render.yaml` automatically
   - Both services deploy simultaneously
   - Takes ~5-7 minutes total

## ✅ Verify Deployment

| Service | URL | Expected Response |
|---------|-----|-------------------|
| Backend | `https://certivault-backend.onrender.com` | JSON response |
| Frontend | `https://certivault-frontend.onrender.com` | Landing page |

## 🔧 Configuration Reference

### render.yaml Structure
```yaml
services:
  - Backend (Python/Flask)
    - Build: pip install requirements.txt
    - Start: python run.py
    - Port: 5000
    
  - Frontend (Node/Next.js)
    - Build: npm install && npm run build
    - Start: npm run start
    - Port: 3000
```

### Environment Variables Set Automatically

**Backend**
- `FLASK_ENV=production`
- `FLASK_DEBUG=False`
- `SECRET_KEY=<auto-generated>`

**Frontend**
- `NEXT_PUBLIC_API_URL=<backend-url>`

## 📊 Builds Timeline

| Stage | Time | Notes |
|-------|------|-------|
| Backend build | 30-60s | pip install |
| Frontend build | 2-3min | npm install + next build |
| Startup | 30-60s | Total cold start ~5-7min |

## 🔑 Important Limits (Free Tier)

- ⏸️ Services spin down after 15 minutes inactivity
- 🥶 First request after inactivity has ~10s cold start
- 📦 SQLite data resets on redeploy (use PostgreSQL for persistence)
- 📝 Limited to 1 free service (use paid for multiple)

## 🛠️ Manual Commands Reference

If deploying separately without render.yaml:

### Backend
```
Environment: Python
Build Command: cd backend && pip install -r requirements.txt
Start Command: cd backend && python run.py
Env Vars:
  - FLASK_ENV=production
  - FLASK_DEBUG=False
  - SECRET_KEY=(auto)
```

### Frontend
```
Environment: Node
Build Command: cd frontend && npm install && npm run build
Start Command: cd frontend && npm run start
Env Vars:
  - NEXT_PUBLIC_API_URL=https://certivault-backend.onrender.com
```

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| Service won't start | Check Logs tab in Render; verify Build/Start commands |
| Frontend can't connect to API | Verify `NEXT_PUBLIC_API_URL` env var has correct backend URL |
| 502 Bad Gateway | Likely backend crashed; check logs |
| Takes >15s to load | Cold start after 15min inactivity; normal on free tier |
| Data disappeared after deploy | Expected on free tier with SQLite; upgrade to PostgreSQL |
